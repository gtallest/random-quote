Project: Random Quote Generator for Free Code Camp

My attempt at the Random Quote Generator project for FCC.

A new random quote will display when the "New quote" button is clicked. The quotes are selected from a predefined array.

Technologies Used:

  • Bootstrap
  • jQuery

Future Features:

  • Play an audio clip of the actual quote when a new quote is displayed.
  • Include a form that allows the user to add their own quotes to the array.
